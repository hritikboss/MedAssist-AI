\# MedAssist AI 🩺



AI-driven symptom checker \& diagnosis assistant built with Python and Streamlit.



\## About

MedAssist AI analyzes user-reported symptoms and suggests possible conditions, along with relevant precautions, medications, diets, and workout recommendations.



\## Tech Stack

\- Python

\- Streamlit

\- SQLite
